4つの設計原則

認知科学とUXデザインを融合した、実践的なアプローチ

⚡

エラーを起こさせない

適切な制約と補助機能で、ミスが発生しにくい設計を実現。 リアルタイムバリデーション、入力マスク、オートコンプリートなど。

🔍

エラーを早期発見

入力直後にフィードバックを提供し、問題を即座に指摘。 具体的なエラーメッセージで、修正方法を明確に。

🔄

簡単リカバリー

自動保存、Undo/Redo、下書き保存で、ミスからすぐに回復。 中断に強く、作業の継続性を保証。

🧠

認知負荷を最小化

情報を記憶させない、見れば分かるUI。 段階的開示、視覚的手がかり、コンテキスト情報の常時表示。
