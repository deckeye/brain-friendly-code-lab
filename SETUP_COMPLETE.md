## 🚀 次のステップ: GitHubへプッシュ

### 手順1: GitHubでリポジトリ作成

1. https://github.com/new にアクセス
2. リポジトリ名: `brain-friendly-code-lab`
3. 説明: `脳に優しい、誰もが使えるUI設計の実践ガイド | エラーゼロ、ストレスフリー、認知負荷最小化`
4. **Public** を選択
5. "Create repository" をクリック

### 手順2: プッシュ

PowerShellまたはGit Bashで実行:

```bash
cd c:/Users/user/Documents/brain-friendly-code-lab

# リモートリポジトリを追加
git remote add origin https://github.com/YOUR_USERNAME/brain-friendly-code-lab.git

# mainブランチに変更
git branch -M main

# プッシュ
git push -u origin main
```

**注意:** `YOUR_USERNAME` を実際のGitHubユーザー名に置き換えてください

### 手順3: GitHub Pagesを有効化

1. GitHubリポジトリの **Settings** タブ
2. 左サイドバーの **Pages**
3. **Source**: `main` ブランチ、`/docs` フォルダ
4. **Save** をクリック

**公開URL (数分後に利用可能):**
```
https://YOUR_USERNAME.github.io/brain-friendly-code-lab/
```
